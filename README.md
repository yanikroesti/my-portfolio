# Yanik — Portfolio

Dark, animated one-page portfolio (React + TypeScript + Tailwind + Framer Motion), personalized from a 3D-creator template.

## Run

```bash
npm install
npm run dev     # http://localhost:4331
npm run build   # typecheck + production build
```
